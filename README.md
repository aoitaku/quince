# quince
[quincite](https://github.com/aoitaku/quincite) for JavaScript/TypeScript.

## Development

```sh
pnpm install
```

## Build

```sh
pnpm run build
```

## Type Check

```sh
pnpm run typecheck
```

## Test

```sh
pnpm run test
```

## Lint

```sh
pnpm run lint
```

## Examples

```sh
pnpm run example:canvas
```

Open http://localhost:5173/ in the browser.
